About EbookLib-JS

EbookLib-JS is a Javascript library for managing EPUB2/EPUB3 and Kindle files. It's forked from the Python library, EbookLib.

Note: Only EPUB creation is currently implemented. Reading EPUB files is a planned feature.

Installation

npm install ebooklib-js
